# LagerPro Frontend

Foreslått frontend-struktur.

## Mapper
- `src/app/` — routing / sider
- `src/components/` — delte UI-komponenter
- `src/features/` — feature-baserte moduler
- `src/lib/` — utilities og helpers
- `src/api/` — API-klienter
- `public/` — statiske filer

## Foreslått stack
- Next.js eller React + Vite
- TypeScript
- Tailwind
- TanStack Query
- React Hook Form
