# LagerPro.Domain

Kjernen i domenet for LagerPro.

## Innhold
- `Common/` — baseklasser og delte typer
- `Enums/` — domenespesifikke enums
- `Entities/` — entiteter og aggregater
- `ValueObjects/` — immutable verdiobjekter
- `Repositories/` — repository interfaces
- `Services/` — domenetjenester
- `Events/` — domeneshendelser
