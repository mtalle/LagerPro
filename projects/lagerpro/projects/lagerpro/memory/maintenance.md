# Weekly memory maintenance

Every week:
- Review `memory/corrections-log.md`
- Remove duplicates
- Keep confirmed rules concise
- Move stale items to `memory/archive/`
- Update `memory/hot/rules.md` and `memory/context/` as needed
- Send Martin a short digest of what changed

Rules:
- Only keep explicit corrections and clear preferences.
- Do not guess from silence.
- If the same correction appears 3 times, ask whether it should become a permanent rule.
